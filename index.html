Table Tags

Tag Function
<table> Defines an HTML table.
<tr> The table row element, is used to add rows to a table before adding table data and table headings.
<td> The table data element can be nested inside a table row element, to add a cell of data to a table.
<thead> TThe table head element defines the headings of table columns encapsulated in table rows.
